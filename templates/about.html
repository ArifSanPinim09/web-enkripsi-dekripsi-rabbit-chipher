About PDFCrypto

A secure, educational implementation of the Rabbit Cipher algorithm for PDF file encryption and decryption.

What is PDFCrypto?

PDFCrypto is a web-based application designed to demonstrate the practical implementation of the Rabbit Cipher algorithm for securing PDF documents. Built as an educational project, it showcases modern cryptographic principles while providing a user-friendly interface for file encryption and decryption.

The application focuses on privacy and security by processing all operations locally, ensuring that your sensitive documents and passwords never leave your control. This approach makes it suitable for both learning purposes and practical use cases where document security is paramount.

The Rabbit Cipher Algorithm

What is Rabbit Cipher?

Rabbit is a high-speed stream cipher designed for high performance in both software and hardware implementations. It was submitted to the eSTREAM project and has been extensively analyzed by the cryptographic community.

Key Features:

  • 128-bit key size
  • High-speed encryption/decryption
  • Stream cipher design
  • Minimal memory footprint

How It Works:

1. Key Setup

Your password is hashed using SHA-256 to create a consistent 128-bit key for the Rabbit cipher.

2. State Initialization

The cipher initializes its internal state using the key through a complex mixing process.

3. Keystream Generation

The algorithm generates a pseudo-random keystream that is XORed with your PDF data.

Security Features

Password Protection

Passwords are never stored and are immediately converted to encryption keys using SHA-256 hashing.

Magic Header Validation

Each encrypted file contains a magic header that validates successful decryption and password correctness.

No Server Storage

All encryption and decryption happens server-side temporarily. Files are processed and immediately deleted.

High Performance

Rabbit cipher's efficient design ensures fast encryption/decryption even for large PDF files.

PDF Integrity

Built-in PDF validation ensures your documents maintain their structure and can be opened normally after decryption.

Open Source

Transparent implementation allows security review and educational study of cryptographic principles.
